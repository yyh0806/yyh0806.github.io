Robust Scene Text Recognition with Automatic Rectification

Abstract

在自然图像中识别文本是一项具有挑战性的任务，存在许多未解决的问题。与文献中的文字不同，自然图像中的文字往往具有不规则的形状，这是由透视变形、弯曲的文字位置等因素造成的。我们提出了一种具有自动校正功能的鲁棒文本识别器，这是一种对不规则文本具有鲁棒性的识别模型。罕见speciallydesigned深层神经网络,它包含一个空间变压器网络(STN)和一个序列识别网络(SRN)。在测试中，首先通过TPS变换对图像进行校正，得到可读性更好的图像用于后续的SRN，该SRN通过序列识别方法识别文本。我们表明，该模型能够识别几种不规则文本，包括透视文本和弯曲文本。RARE是端到端的，只需要图像和相关的文本标签，这使得在实际系统中培训和部署模型非常方便。在几个基准测试中达到的最先进或高度竞争的性能很好地证明了所提出的模型的有效性。

Introduction

在自然场景中，文字出现在各种物体上，如路标、广告牌、产品包装等。它承载着丰富而高级的语义信息，这些语义信息对图像的理解非常重要。识别图像中的文本有助于许多真实世界的应用程序，如地理定位、无人驾驶汽车和基于图像的机器翻译。基于这些原因，场景文本识别引起了社会各界的极大兴趣[28,37,15]。尽管光学字符识别(OCR)[26]的研究已经成熟，但是识别自然图像中的文本，而不是扫描文档，仍然是一个挑战。场景文本图像在照明、运动模糊、文本字体、颜色等方面表现出很大的变化。此外，文字在自然条件下可能有不规则的形状。例如，一些场景文本为视角文本[29]，它是由相机侧视角度引起的;有些有曲线形状，这意味着它的文字是沿着曲线而不是直线放置的。我们称这种文本为不规则文本，而常规文本是横向和正面的。

通常，文本识别器在其输入图像中包含严格限制的规则文本时工作得最好。这促使我们在识别之前应用空间变换，以便将输入的图像校正为识别器更容易识别的图像。本文提出了一种对不规则文本具有鲁棒性的识别方法。具体地说，我们构造了一个深度神经网络，它结合了空间变压器网络(STN)和序列识别网络(SRN)。模型的概述如图1所示。

图1所示。RARE的原理概述，它由一个空间变压器网络(STN)和一个序列识别网络(SRN)组成。STN将输入图像转换为校正后的图像，而SRN识别文本。两个网络由[22]反向传播算法共同训练。虚线代表了反向传播的梯度的流动。

在STN中，输入图像在空间上转换为校正后的图像。理想情况下，STN生成一个包含规则文本的图像，对于SRN来说，这是比原始文本更合适的输入。该转换是一个thin-plate-spline(TPS)转换，其非线性允许我们纠正各种类型的不规则文本，包括透视和弯曲文本。TPS变换由一组基准点(fiducial points)构成，其坐标由卷积神经网络回归。

在包含正则文本的图像中，字符沿水平线排列。它与序列信号有一些相似之处。基于此，我们为SRN构造了一个基于注意力的模型[4]，它可以用序列识别方法识别文本。SRN包括一个编码器和一个解码器。给定一个输入图像，编码器生成一个序列特征表示，这是一个序列的特征向量。该解码器通过解码由其每一步的注意机制所决定的相关内容，在输入序列上递归地生成符合条件的字符序列。

我们证明，适当的初始化，整个模型可以训练端到端。因此，对于STN，我们不需要标注任何几何ground truth，即TPS基准点的位置，而是让它的训练由SRN反向传播的误差差来监督。在实践中，这种训练最终使STN倾向于生成包含规则文本的图像，这是SRN所希望的输入。

本文的贡献有三个方面:首先，我们提出了一种新的场景文本识别方法，该方法对不规则文本具有鲁棒性。其次，我们的模型使用基于注意力的模型扩展了STN框架[18]。原来的STN只在普通卷积神经网络上测试。第三，我们的模型在SRN的编码器中采用了卷积循环结构，是基于注意力的[4]模型的一个新版本。

Related Work

近年来，大量关于场景文本识别的文献相继发表。[40,44]给出了全面的调查。在传统的方法中，很多采用自底向上的方法，首先使用滑动窗口检测单个字符[36,35]，连接组件[28]，或霍夫投票[39]。然后通过动态规划、词汇搜索[35]等方法将检测到的字符集成到单词中。其他工作采用自顶向下的方法，即直接从整个输入图像中识别文本，而不是检测和识别单个字符。例如，Almazan等人[2]提出从输入图像中预测标签嵌入向量。Jaderberg等人使用90k类卷积神经网络对[17]地址文本进行识别，其中每个类对应一个英语单词。在[16]中，构造了一个带有结构化输出层的CNN，用于无约束文本识别。最近的一些工作将该问题建模为序列识别问题，其中文本用字符序列表示。Su和Lu[34]提取序列图像表示，即HOG[10]描述符的序列，并用递归神经网络(RNN)预测对应的特征序列。Shi等[32]提出了一种结合CNN和RNN的端到端序列识别网络。我们的方法也采用了序列预测方案，但我们进一步考虑了不规则文本的问题。

不规则文本虽然在场景文本检测和识别任务中很常见，但对不规则文本的明确处理相对较少。Yao等[38]首先提出了多面向文本检测问题，并通过精心设计旋转不变的区域描述符来处理该问题。Zhang等人[42]提出了一种利用文本的low-rank结构的字符校正方法。Phan等人提出通过SIFT[23]描述符匹配显式地纠正透视失真。上述工作给这个问题带来了深刻的见解。然而，大多数方法只处理一种特殊设计方案的不规则文本。我们的方法统一地校正了几种不规则的文本。此外，在矫正过程中不需要额外的注释，因为STN在训练时是由SRN监督的。

Proposed Model

在本节中，我们将阐述我们的模型。总的来说，模型取一个输入图像I，输出序列l = (l1,…,lT)\left(l_{1}, \ldots, l_{T}\right)(l1​,…,lT​)其中，lTl_{T}lT​是第T个字符，T是可变的字符串长度。

Spatial Transformer Network

STN通过预测的TPS变换将输入图像III转换为校正后的图像I′I^{\prime}I′。它遵循[18]中提出的框架。如图2所示，它首先通过定位网络预测一组基准点。然后，在网格生成器内部，从基准点计算TPS变换参数，在III上生成一个采样网格，采样器同时取网格和输入图像，对网格点进行采样，生成一个校正后的图像I′I^{\prime}I′。

STN的一个特殊性质是它的采样器是可微的。因此，一旦我们有了可微的定位网络和可微的网格生成器，STN就可以反向传播误差微分并得到训练。

TPS转换的灵活性允许我们将不规则的文本图像转换为包含规则文本的校正图像。在图4中，我们展示了一些常见的不规则文本类型，包括a)松散边界文本，这是由于文本检测不完善造成的;b)多方向文字，相机非横向视图造成;c)透视文本，由相机侧视角度引起;d)弯曲文本，一种常见的艺术风格。STN能够对包含这些不规则文本类型的图像进行校正，使它们对下面的识别器更具可读性。

Sequence Recognition Network

由于目标词本质上是字符的序列，我们将识别问题建模为序列识别问题，并使用序列识别网络来解决它。SRN的输入是一个经过校正的图像I′I^{\prime}I′，理想情况下包含一个从左到右水平书写的单词。我们从I′I^{\prime}I′中提取一个序列表示，并从中辨认出一个单词。

在我们的模型中，SRN是一个基于注意力的模型[4,8]，它直接从输入图像中识别序列。SRN包括一个编码器和一个解码器。编码器从输入图像I′I^{\prime}I′中提取序列表示。解码器通过解码每一步处理的相关内容，递归地生成一个以顺序表示为条件的序列。
