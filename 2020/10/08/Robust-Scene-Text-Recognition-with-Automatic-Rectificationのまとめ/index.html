Robust Scene Text Recognition with Automatic Rectification

背景

Recognizing text in natural images is a challenging task with many unsolved problems. Different from those in documents, words in natural images often possess irregular shapes, which are caused by perspective distortion, curved character placement, etc. We propose RARE (Robust text recognizer with Automatic REctification), a recognition model that is robust to irregular text. RARE is a specially　designed deep neural network, which consists of a Spatial Transformer Network (STN) and a Sequence Recognition Network (SRN).

自然画像内のテキストを認識することは、多くの未解決の問題を伴う困難な作業です。 自然画像の単語は、文書とは異なり、遠近法による歪みや文字の配置の湾曲などにより不規則な形状をしていることが多い。我々は、不規則なテキストに対して頑健性を持つ認識モデルとして、自動補正機能を有するロバストなテキスト認識器を提案した。RAREは、空間変圧器ネットワーク(STN)とシーケンス識別ネットワーク(SRN)を組み合わせる深層ニューラルネットワークの特殊設計。

解決する課題

不規則なテキスト（irregular text）の識別

解決策

RARE (Robust text recognizer with Automatic REctification)

Spatial Transformer Network (STN)

STNは予測されたTPS変換で、入力画像を修正画像に変換します。

Sequence Recognition Network (SRN)

SRNは、エンコーダーとデコーダーで構成される注意ベースのモデル（attention-based model）。

入力(修正画像)からシーケンス（sequence）を識別します。

補足資料

薄板スプライン Thin-Plate-Spline (TPS)

薄板スプラインは、データの補間と平滑化のためのスプラインベースの手法です。それらはDuchonによって幾何学的設計に導入されました。それらは、多調和スプラインの重要な特別な場合です。ロバストポイントマッチングは一般的な拡張機能であり、TPS-RPMアルゴリズムとしてすぐに知られています。

Sampler

作成された画像VVVはバイリニア補間（Bilinear interpolation）で修正画像I′I^{\prime}I′を生成する。

バイリニア補間では求める位置(x,y)の周辺の２×２画素（４画素）を使って、輝度値を直線的に補間して、輝度値を求めます。

Sequence Recognition Network (SRN)

Encoder : Convolutional-Recurrent Network

The encoder uses several convolution layers (ConvNet) and a two-layer BLSTM network to extract a sequential representation (h) for the input image.

convolution層とBLSTMネットワークを使って、入力画像の特徴情報(sequential representation)を検出する

Decoder : Recurrent Character Generator

The decoder recurrently generates a sequence of characters, conditioned on the sequence produced by the encoder。

デコーダーは文字のシーケンスを繰り返し生成します。
