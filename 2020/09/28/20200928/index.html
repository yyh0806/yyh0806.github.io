测试GANomaly

目标

用syaft数据集进行测试，因为washer的数据太少很容易达到1。

为了使用syaft，删除IMAGE_SIZE。使用IMAGE_RESIZE_HEIGHT和IMAGE_RESIZE_WIDTH生成模型。

删除IS_USE_SSIM，使用CON_LOSS_PATTERN

修改model，原本的模型不支持长宽不一致的输入。

对三个参数进行测试，看看对生成结果的影响。

  1. 图像大小 64x128/ 128x256 / 256x512
  2. con_loss L1 loss / SSIM loss / L1 loss + SSIM loss
  3. NZ 10 / 100 / 500 /
