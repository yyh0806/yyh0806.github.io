GANomaly

原理

概要

GANomaly ではGANの敵対的な構造に加えて AutoEncoder と呼ばれるネットワークを導入します。

論文GANomaly: Semi-Supervised Anomaly Detection via Adversarial Training

GANomalyはSemi-Supervisedではないと思います。

そういえばタイトルに Semi-Supervised と入っているが、どの辺りが semi-supervised なのだろうか。
GAN で semi-supervised と言ったら典型的には discriminator の出力が classification のカテゴリとなるようにして学習にカテゴリ情報を使うとかだと思っていたが、GANomaly ではラベル情報は使ってないんだよな。

アーキテクチャ

AutoEncoderは画像の圧縮と復元を行うネットワークであり、ノイズの除去や次元の圧縮などに利用されます。AutoEncoderとGANの構造を組み合わせ、潜在空間をAutoEncoderの圧縮された特徴表現によって構成したのが GANomaly です。

GAN を使った異常検知のモデルで、AutoEncoder 型で Encoder-Decoder-Encoder の構造を取る。

他のモデルとの比較

名前 アーキテクチャ
AutoEncoder AutoEncoder
AnoGAN AnoGANのアーキテクチャはDCGANなどのベーシックなGAN構造をしてる。
Efficient GAN Efficient-GAN は encoderが加わり、encoder と generator を同時に学習させる。

目的関数

L=wadvLadv+wconLcon+wencLenc\mathcal{L}=w_{a d v} \mathcal{L}_{a d v}+w_{c o n} \mathcal{L}_{c o n}+w_{e n c} \mathcal{L}_{e n c} L=wadv​Ladv​+wcon​Lcon​+wenc​Lenc​

loss としては以下の三つになっている。

  • Ladv:\mathcal{L}_{a d v}:Ladv​: discriminator の出力値の l2 距離 (adversarial loss)
  • Lcon:\mathcal{L}_{\text {con}}:Lcon​: 入力画像と再構築した画像の l1 距離(contextual loss)
  • Lenc:\mathcal{L}_{\text {enc}}:Lenc​: 入力画像と再構築した画像を encode した潜在空間での l2 距離(encoder loss)

contexual lossについて、

adversarialなlossによりrealな画像が再構築されるようになるが、それが元画像っぽい保証はない。そこで元画像っぽいものを再構築するよう、両者でL1をとる。

画像と画像の類似性比較方法を考慮して、contexual lossにSSIM lossを追加しました。
